## Tips & Tricks üí°

- **Multiple Elements**: You can add multiple images, text, and stickers
- **Layering**: Elements added later appear on top
- **Print Area**: The dashed border shows the safe print area
- **Selection**: Click outside elements to deselect
- **Model Change**: Changing phone models will reset your design

## Troubleshooting

**Image not uploading?**
- Check file size (max 5MB)
- Ensure format is JPG, PNG, or GIF

**Can't move elements?**
- Make sure element is selected (purple ring)
- Click directly on the element

**Design looks different in preview?**
- Preview shows scaled version
- Final product will match canvas design

**Elements disappearing?**
- They might be outside the visible area
- Try changing phone model back and forth to reset
